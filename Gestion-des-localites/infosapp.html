DESCRIPTION DE L'APPLICATION

Cette application web vise à mettre en place, via des formulaires web, de :

pouvoir gérer (ajout, suppression, modification) les différentes localités d’une commune ;

créer des équipes et ses membres ;

pourvoir importer un fichier (Excel ou CSV ou tout autre type de fichier qui vous convient) contenant la liste des agents vers la table AGENT ;

L’accès à l’application doit être sécurisé via un login et un mot de passe qui seront lus depuis une table « UTILISATEUR » de la base de données.
